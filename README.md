Developed a full-stack food ordering system that enables customers to order meals from a standard menu or a dynamically updated Menu of the Day, with seamless third-party delivery integration.

Highlights
* Designed and implemented menu browsing, cart management, and order processing features.
* Integrated external delivery services such as Uber (Uber Eats) and DoorDash for home delivery fulfillment.
* Implemented secure payment processing where users are charged only for the food ordered, while delivery fees are handled by the delivery provider.
* Built RESTful APIs to manage orders, payments, and delivery coordination.
* Ensured proper authentication and role-based authorization for secure transactions.
